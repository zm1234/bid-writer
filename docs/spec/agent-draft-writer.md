# Agent规范 - 初稿撰写

## 概述

根据PRD和规则集，生成标书初稿。

## 输入

- PRD（项目定义）
- 规则集（格式要求+风格档案）
- 参考论文/背景材料

## 输出

完整的标书初稿，包含以下章节：

1. 项目摘要
2. 研究背景与意义
3. 研究目标与内容
4. 研究方法与技术路线
5. 预期成果
6. 研究基础与条件
7. 进度安排
8. 经费预算

## 核心Prompt模板

```
# 标书初稿撰写任务

## 项目信息
- 项目标题：{title}
- 研究领域：{field}
- 资助类型：{fundType}

## PRD
{PRD_CONTENT}

## 绝对规则（必须遵守）
{ABSOLUTE_RULES}

## 风格档案
{STYLE_PROFILE}

## 参考材料
{REFERENCE_MATERIALS}

请根据以上信息，按照中国科研项目标书的规范格式，撰写完整的初稿。

注意事项：
1. 严格遵守字数限制
2. 保持学术严谨性
3. 突出创新点
4. 明确研究可行性
5. 量化预期成果
```

## 状态管理

- `drafting`: 撰写中
- `completed`: 初稿完成
- `reviewing`: 提交审核

## 迭代支持

初稿撰写支持增量更新：
- 修改特定章节
- 根据审核反馈重写部分内容
- 补充缺失信息
